# 🦙 llama-ide

A powerful IDE with AI assistance integrated with Ollama and cloud LLM providers.

## Features
- Code editor with AI autocomplete
- Terminal integration
- File tree and tab management
- Chat playground with streaming responses
- Support for local (Ollama) and cloud models
- Dark/light theme toggle

## Quick Start

1. Install dependencies:
   ```bash
   make install-all
   ```

2. Start development servers:
   ```bash
   make dev
   ```

3. Access the application:
   - Frontend: http://localhost:5173
   - Backend: http://localhost:8000

## Requirements
- Node.js >= 16
- Python >= 3.8
- Ollama (for local models)
